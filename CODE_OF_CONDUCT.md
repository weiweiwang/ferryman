# Code of Conduct

## Our Standard

Contributors should help keep Ferryman collaborative, respectful, and useful.
We expect participants to:

- communicate professionally and constructively
- assume good intent while addressing technical disagreements directly
- avoid harassment, intimidation, discrimination, or personal attacks
- respect privacy, security, and responsible disclosure boundaries

## Unacceptable Behavior

Examples of unacceptable behavior include:

- abusive or demeaning language
- targeted harassment
- publishing private information without permission
- intentionally introducing malware, credential leaks, or destructive changes

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict
participation for behavior that violates this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the
maintainer or owning organization instead of escalating it in public threads.
